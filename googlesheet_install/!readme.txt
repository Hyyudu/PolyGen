Для работы PolyGen с Google Docs необходимо выполнить следующие действия:
1. Поставить себе Python 3 (работа с версиями Python 2.x не гарантируется)
2. Скопировать файл python-pyOpenSSL.ymp в папку lib внутри папки Python 3
3. Запустить install_googlesheets.bat. Если он выдает ошибку вида "pip не является внутренней или внешней командой, исполняемой программой или пакетным файлом.", нужно перенести install_googlesheets.bat в папку, где находится pip.exe (обычно python3/Scripts) и запустить там.
Эти действия надо выполнить один раз, чтобы к вашему Python были подключены нужные компоненты.
Теперь, чтобы Python мог получить доступ к Google Sheets, нужно дать доступ для email polygraph.gen@gmail.com.
